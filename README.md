# una-blazor-lista14 - Meu Currículo Blazor

Este projeto é uma aplicação web desenvolvida em Blazor para a **Lista de Exercícios XIV** da disciplina de Interação Humano Computador e UX. O objetivo principal é a personalização e profissionalização de um currículo digital, aplicando conceitos de design e usabilidade.

## 🚀 Guia de Execução
Para rodar este projeto localmente via terminal, siga os passos abaixo:

1.  Certifique-se de ter o [.NET SDK](https://dotnet.microsoft.com/download) instalado.
2.  Abra o terminal na pasta raiz do projeto.
3.  Execute o comando para rodar a aplicação:
    ```bash
    dotnet run
    ```
    *Dica: Use `dotnet watch` para habilitar o Hot Reload durante o desenvolvimento.*

4.  Acesse a URL informada no terminal (geralmente `http://localhost:5000` ou `https://localhost:5001`).
